NAME

o2b − Symbol file browser

SYNOPSIS

o2b [option]... module

DESCRIPTION

o2b reconstructs a printable representation from a module’s symbol file. The output is similar to Modula-2 definition files.

OPTIONS

--expand, -x

Expand extended records. Print for each record type its base types, which fields were inherited from them, and what type bound procedures were (re-)defined.

--external, -e

Print external names. For external modules the calling convention, the external library name, and (if they differ from the Oberon-2 names) external names for variables and procedures are displayed.

--help, -h

Give a summray on how to call o2b.

--version, -V

Print browser version.

--redir <file>

Use <file> as redirection table (instead of the default ˜/.o2c.red).
