NAME

o2whereis − locate files used or generated by o2c

SYNOPSIS

o2whereis [-r] file

DESCRIPTION

o2whereis searches the paths given in ˜/.o2c.red for the file file. How the search is done is described in o2c(1). The full path of the first file found is printed to stdout.
o2whereis is used by the Oberon-2 Emacs mode to locate source and symbol files.

OPTIONS

-r

If just a RCS file is found, print the full path of the RCS file. The default is to print the name of the associated working file.

--redir <file>

Use <file> as redirection table (instead of the default ˜/.o2c.red).

FILES

˜/.o2c.red path list
